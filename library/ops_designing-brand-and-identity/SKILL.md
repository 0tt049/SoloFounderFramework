# Designing Brand and Identity

## Goal
To elevate layout structures for optimal conversion performance and craft authoritative, psychologically aligned brand color guidelines tailored to product domains.

## When to Use This Skill
- When conducting UX audits or seeking conversion optimization on active landing pages.
- When selecting typography, brand assets, or layout color palettes for specialized client sectors.

### Do Not Use This Skill For:
- Writing low-level CSS stylesheets/scripts or generating raw SVG images.

## Plan-Validate-Execute Workflow
1. **Plan (Target Audience & Structure Audit):**
   - Identify target audience characteristics (e.g., trust-oriented professions vs. creative consumers).
   - Trace the visual flow of landing pages from top (above-the-fold) to footer.
2. **Validate (Enforce Design Guardrails):**
   - Guarantee conversion-oriented rules are locked: All sales landing pages require prominent CTAs (Call to Action) above the fold, supplemented with social proof (customer quotes, peer reviews, badges).
   - Filter palette choices based on professional archetypes: Traditional B2B or legal software mandates conservative, trustworthy tones (e.g., deep Navy blue, slate gray, solid white, dark forest green). Explicitly blacklist neon or overly playful vibes.
3. **Execute (Document Rules & Palette):**
   - Provide precise layout and content enhancements targeting UX best-practices.
   - Explain the color psychology behind recommendations clearly (e.g. Navy represents security/competence; slate represents balance).

## Design Rules & Guidelines

### Conversion Layout Norms
- Landing pages must declare clear, primary CTA text in the first scrollable panel (above-the-fold).
- Highlight credibility markers (testimonials, ratings, user stats) directly beneath CTAs or alongside feature matrices.

### Palette Psychology Boundaries
- Target audience shapes the spectrum: Trustworthiness requires high contrast, neutral, and muted configurations. Navy, Charcoal, Slate, and dark greens are standard anchors.
- Overly saturated accents (hot pink, high-activity neons) are restricted from formal enterprise contexts.

## Example Implementation Checklist
Copy this checklist during design audit sessions:

- [ ] Survey layout zones: verify presence of above-the-fold CTA.
- [ ] Recommend strategic placements for user testimonials or logo clouds.
- [ ] Align style recommendations with design-focused and conversion-oriented heuristics.
- [ ] Build color suites utilizing navy, slate, dark green, or white base tones for corporate profiles.
- [ ] Detail the visual psychology associated with each proposed palette selection.
- [ ] Rigorously vet palettes to exclude neon or overly playful shades.
