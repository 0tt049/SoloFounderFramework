# Optimizing SEO Content

## Goal
To audit written materials and optimize target keywords, meta descriptions, readability grade levels, and technical search snippets supporting modern brand visibility.

## When to Use This Skill
- When conducting SEO optimization on blog entries, landing pages, or web copy.
- When drafting promotional content targeting organic traffic.
- When auditing existing copy for keyword density, structural clarity, or formatting gaps.

### Do Not Use This Skill For:
- High-level server-side tech SEO configuration (e.g. sitemap routing, robots.txt, CDN caching, SSL handshake).

## Plan-Validate-Execute Workflow
1. **Plan (Target Audit):**
   - Identify the primary target keywords, secondary terms (LSI variations), and the target customer's search intent.
2. **Validate (Readability & Density Checking):**
   - Measure Keyword Density (maintain between 1% and 2%; flag anything above 3% as keyword stuffing).
   - Ensure the primary keyword is located in Title/H1, the first 100 words, and at least one H2 heading.
   - Calculate Flesch Reading Ease Grade level (target eighth-grade level, score 60-70).
   - Trace sentence structures and passive voice levels (keep passive voice under 10%).
3. **Execute (Render Optimization Results):**
   - Provide an SEO Performance Scorecard (0-100), word counts, and Grade level.
   - List keyword/semantic gap additions clearly.
   - Deliver optimized URL slug, Meta Title (50-60 chars including primary keyword), and Meta Description (150-160 chars ending with a clear CTA).
   - Draft before-and-after rewrite samples showing direct, scannable format improvements and featured snippet setups.

## Design Rules & Guidelines

### Layout & Readability Constraints
- Headers must follow a logical nesting hierarchy (one H1, nested H2, H3).
- Break paragraphs down to a maximum of 150 words (2-3 sentences) to facilitate scanability on mobile devices.
- Include structured list or FAQ elements to capture Google's Answer Boxes/Featured Snippets.

### Copywriting Rules
- Ensure content serves human readers first, placing technical keywords organically rather than forcing irrelevant placements.
- Avoid clichés, empty marketing fluff, and over-sensationalized CTA hooks.

## Example Implementation Checklist
Copy this checklist during copy auditing operations:

- [ ] Check primary keyword placement in top structural tags (H1, intro, H2).
- [ ] Measure density ratio against the 1-2% baseline.
- [ ] List LSI keyword gaps matching search semantic context.
- [ ] Calculate the readability ease score and grade level.
- [ ] Draft compliance-perfect meta titles and CTA-rich descriptions.
- [ ] Supply before-and-after markdown transformations optimized for rich snippets.
